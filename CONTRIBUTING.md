# Contributing

Thank you for your interest in LM Gate!

## How to Contribute

At this time, we do **not** accept pull requests. The best way to contribute is by submitting questions, bug reports, and feature requests using the links below:

- [Ask a Question](https://github.com/hkdb/lmgate/issues/new?template=question.yml)
- [Report a Bug](https://github.com/hkdb/lmgate/issues/new?template=bug.yml)
- [Request a Feature](https://github.com/hkdb/lmgate/issues/new?template=feature_request.yml)

For security vulnerabilities, please see our [Security Policy](SECURITY.md).
